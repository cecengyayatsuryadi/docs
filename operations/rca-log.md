# Root Cause Analysis Log

> Deep analysis of significant failures. Not for quick fixes — only for failures worth understanding thoroughly.

**Last updated:** YYYY-MM-DD  
**Template:** [templates/rca-template.md](../templates/rca-template.md)

---

## When to Add an Entry

- A production bug took more than 30 minutes to diagnose
- A failure cascaded across components
- The same type of failure has happened before
- The root cause was non-obvious (not a simple typo)
